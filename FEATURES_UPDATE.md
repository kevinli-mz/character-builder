# 新功能更新说明

本次更新为应用添加了以下新功能：

## 1. 卡牌设计功能

### 功能描述
- 用户可以装备卡牌，卡牌置于角色后面，角色在卡牌上半部分
- 角色可以选择不同的形状遮罩（方形、圆形、菱形）
- 卡牌下半部分有透明文本框可以添加：
  - 用户的名称（大标题）
  - 一句用户想写下的一句话（正文小字）
- 管理员可以在后台上传不同的卡片设计
- 用户选择【下载角色】的时候：
  - 若装备了卡片，则下载卡片 + 角色的长方形图片（3:4比例）
  - 若没装备卡片，则下载用户only的方形图片（1:1比例）

### 实现细节
- 数据库：新增 `cards` 表存储卡牌信息
- API：`fetchCards()`, `uploadCard()`, `deleteCard()`
- 组件：`Canvas` 组件支持卡牌背景和文本渲染
- 下载：根据是否装备卡牌生成不同尺寸的图片

## 2. 用户管理界面

### 功能描述
- 新增用户管理界面，用户通过点击右上角的头像图标进入
- 显示用户邮箱和预设列表

### 实现细节
- 新增 `UserProfile` 组件
- 路由：通过 `#profile` hash 路由访问
- 入口：Configurator 右上角头像按钮

## 3. 用户预设功能

### 功能描述
- 用户可以将特定的组合保存为自己的预设，并给他们命名
- 下载角色的按钮旁边增加一个"保存预设"按钮
- 预设保存后有成功提示
- 用户可以在用户管理界面中看到自己所有保存的预设
- 可以加载预设或删除预设

### 实现细节
- 数据库：新增 `presets` 表存储预设信息
- API：`savePreset()`, `getUserPresets()`, `deletePreset()`, `loadPreset()`
- 预设包含：角色状态、卡牌ID、遮罩形状、文本标题和正文

## 4. 卡牌3D互动效果

### 功能描述
- 鼠标和卡牌互动的时候，就会产生3D perspective tilt的互动效果

### 实现细节
- 使用 CSS `perspective` 和 `transform` 实现
- 鼠标移动时计算相对位置并应用旋转
- 仅在装备卡牌时启用

## 5. 管理员卡牌管理

### 功能描述
- 管理员可以在后台管理卡牌
- 上传新卡牌（支持设置默认卡牌）
- 查看所有卡牌
- 删除卡牌

### 实现细节
- AdminDashboard 新增"卡牌管理"标签页
- 支持拖拽上传
- 显示卡牌预览和默认标识

## 使用说明

### 用户端
1. **装备卡牌**：在左侧控制面板选择卡牌
2. **选择遮罩**：装备卡牌后可以选择形状遮罩（方形/圆形/菱形）
3. **编辑文本**：点击"编辑文本"按钮，输入标题和正文
4. **保存预设**：点击"保存预设"按钮，输入名称保存
5. **查看预设**：点击右上角头像，进入用户管理界面查看所有预设
6. **下载**：根据是否装备卡牌，下载不同尺寸的图片

### 管理员端
1. **上传卡牌**：进入管理面板 -> 卡牌管理 -> 上传新卡牌
2. **设置默认**：上传时勾选"设为默认卡牌"
3. **删除卡牌**：在卡牌列表中点击删除按钮

## 注意事项

1. 卡牌图片建议使用垂直矩形（3:4比例）以获得最佳效果
2. 预设会保存当前的所有设置，包括角色、卡牌、遮罩和文本
3. 删除卡牌不会影响已保存的预设，但预设中的卡牌将无法显示
4. 3D tilt效果仅在桌面端鼠标悬停时生效
